# Gong_cx 导航站源码

#### 介绍
一个很废物的导航站代码，甚至只能手动修改 HTML 内容

这是一个非常不完善的源码，只能自己修改 HTML 内容，背景动画使用开源项目实现，在此感谢。

#### 安装教程
1. 下载最新的发行版
2. 修改 HTML 内容（此处需要您有基本 HTML 开发知识）
3. FTP 上传到您的网站运行目录
4. 给本仓库来个 Star 并 Fork 

#### 其他
jump.html 所采用的跳转方式，为延时跳转，你可以随意修改延长时间，让访客多看会 Loading...
